About This Project

This Sentiment Analysis system was developed as a final year project to analyze the emotional tone behind written text. It uses a deep learning model powered by a Bidirectional LSTM (Long Short-Term Memory) network, enhanced with pre-trained GloVe word embeddings for improved language understanding.

The model was trained on the IMDB movie review dataset, along with additional manually labeled samples to recognize both subtle and strong emotional cues. Users can test the sentiment of custom text or even analyze real-time content from Reddit using integrated APIs.

Features:

  • Accurate sentiment prediction (Positive, Negative, Neutral)
  • Preprocessing and text cleaning pipeline
  • Integration with Reddit for real-world sentiment tracking
  • Responsive Flask-powered web interface
  • Built using TensorFlow, Keras, Pandas, and Matplotlib

Purpose:

The aim of this project is to demonstrate how machine learning can be applied to understand and interpret natural language — offering insights into public opinion, feedback, and trends.
